    Welcome to the Web Bomic Collect Site

    This site will give you everything you need to know to get up and running as smoothly as possible on Web Bomic Collect. The key features of Web Bomic Collect are:

    • Web Bomic Collect allows Substance Misuse services around the UK to spend less time writing case notes and in turn provides a more efficient way of working

    • Capture data in outreach setting securely

    • Reduction in transcription errors as all data is uploaded electronically

    • Eradicates inputting data into Web Bomic Core system from paper notes

    • Less money spent on travel expenses going to and from the office

    • More secure method of data capture

    • All data is fully encrypted and no identifiable information is on the tablet

    • Tablet can only be accessed by a specific user with a username and password

    • Allows services to see more clients in less time, and ensure time with the client is of a better quality
